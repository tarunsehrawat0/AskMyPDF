# AskMyPDF - Research Assistant

A powerful PDF research assistant that lets you upload multiple PDF documents and ask questions about their content using AI-powered search and answer generation.

## Features

 🤖 **AI-Powered Answers**: Get intelligent answers generated by OpenAI
- 📄 **Source Tracking**: See which documents provided the answers with source tags
- 🎨 **Professional UI**: Modern, responsive interface built with Tailwind CSS
- ⚡ **Fast Search**: FAISS-powered vector search for instant results
- 💾 **Persistent Storage**: Indexed documents persist between sessions

## Installation

1. **Install dependencies**:
```bash
pip install -r requirements.txt
```

2. **Set up OpenAI API key**:
   - Copy `.env.example` to `.env`
   - Add your OpenAI API key to `.env`:
   ```
   OPENAI_API_KEY=your_actual_api_key_here
   ```

## Usage

1. **Start the FastAPI health app**:
```bash
python app.py
```

   Or, with Uvicorn directly:
```bash
uvicorn app:app --host 0.0.0.0 --port 8000
```

2. **Open your browser**:
   Navigate to `http://localhost:8000`

3. **Run the full Streamlit interface**:
```bash
python -m streamlit run streamlit_app.py --server.port 8501 --server.address 0.0.0.0
```

4. **Open the Streamlit UI**:
   Navigate to `http://localhost:8501`

3. **Upload PDFs**:
   - Click the upload area or drag and drop PDF files
   - Multiple files supported
   - Documents will be automatically indexed

4. **Ask questions**:
   - Type your question in the query box
   - Click "Search & Generate Answer" or press Enter
   - View the AI-generated answer with source document tags

## API Endpoints

- `POST /upload` - Upload a PDF file
- `POST /ask` - Ask a question about indexed documents
- `GET /documents` - List all indexed documents
- `POST /clear` - Clear all indexed documents

## Technology Stack

- **Backend**: Flask (Python)
- **PDF Processing**: PyPDF2, pdfplumber
- **Text Embeddings**: Sentence Transformers (all-MiniLM-L6-v2)
- **Vector Search**: FAISS
- **AI Generation**: OpenAI GPT-4o mini
- **Frontend**: HTML, Tailwind CSS, JavaScript
- **Icons**: Font Awesome 6
- **Fonts**: Inter (Google Fonts)

## How It Works

1. PDFs are uploaded and text is extracted using pdfplumber
2. Text is chunked into smaller segments (500 words with overlap)
3. Each chunk is converted to a vector embedding using Sentence Transformers
4. When you ask a question, it's also converted to an embedding
5. FAISS finds the most similar chunks using cosine similarity
6. Gemini AI generates an answer based on the relevant chunks
7. The answer is returned with source document references

## Notes

- First run will download the Sentence Transformer model (~100MB)
- Requires an OpenAI API key for answer generation (get one at [platform.openai.com](https://platform.openai.com))
- Indexed data is persisted in `metadata.json`
- PDFs are stored in the `pdfs/` directory
- The UI is fully responsive and works on mobile devices
